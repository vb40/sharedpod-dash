
# SharedPod Dashboard

A modern dashboard for team and project management built with React, TypeScript, and Tailwind CSS.

## Features

- Team Management
- Project Tracking
- Ticket Management
- Performance Analytics
- Certification Tracking
- Holiday Calendar
- Attendance Tracking

## Getting Started

1. Clone the repository
2. Install dependencies:
```sh
npm install
```
3. Start the development server:
```sh
npm run dev
```

## Technologies Used

- React
- TypeScript
- Tailwind CSS
- Recharts
- Tanstack Query
- Shadcn/ui
